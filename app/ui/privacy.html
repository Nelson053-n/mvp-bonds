Политика конфиденциальности

Последнее обновление: 13 марта 2026 г.

1. Оператор персональных данных

Сервис Bond AI (далее — «Сервис», «мы»), доступный по адресу bondai.ru, осуществляет обработку персональных данных пользователей в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

Используя Сервис, вы даёте согласие на обработку ваших данных на условиях настоящей Политики.

2. Какие данные мы собираем

  • Логин (имя пользователя) — для идентификации аккаунта
  • Пароль — хранится исключительно в виде хеша (bcrypt), исходный пароль не сохраняется
  • Email — при указании, для восстановления пароля
  • Данные портфеля — тикеры, количество бумаг, цены покупки
  • Настройки уведомлений — Telegram chat ID, если вы подключили бот
  • Техническая информация — IP-адрес, User-Agent, время запросов (для обеспечения безопасности)

3. Цели обработки данных

Мы обрабатываем данные исключительно для:

  • Предоставления функций Сервиса — учёт портфеля, расчёт доходности, AI-подбор
  • Отправки уведомлений — купонные выплаты, изменения рейтингов, ценовые алерты (через Telegram)
  • Обеспечения безопасности — защита от несанкционированного доступа, rate-limiting
  • Улучшения Сервиса — агрегированная аналитика использования (Яндекс.Метрика)

4. Хранение данных

Данные хранятся на серверах на территории Российской Федерации. Мы применяем следующие меры защиты:

  • Шифрование паролей (bcrypt с солью)
  • JWT-аутентификация с ограниченным сроком действия
  • HTTPS-соединение для всех запросов
  • Регулярное резервное копирование базы данных

Данные хранятся в течение всего периода использования аккаунта. При удалении аккаунта все персональные данные удаляются.

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши персональные данные третьим лицам. При этом для работы Сервиса используются внешние API:

  • Московская биржа (MOEX ISS API) — мы запрашиваем публичные рыночные данные; ваши персональные данные не передаются
  • Anthropic / OpenAI API — при AI-анализе передаются обезличенные параметры портфеля (тикеры, суммы) без персональных данных пользователя
  • Telegram Bot API — если вы подключили уведомления, для отправки сообщений используется ваш chat ID
  • Яндекс.Метрика — для сбора агрегированной статистики посещений

6. Права пользователя

В соответствии с законодательством РФ вы имеете право:

  • Запрашивать информацию о ваших персональных данных, которые мы обрабатываем
  • Требовать исправления неточных данных
  • Требовать удаления ваших данных (удаление аккаунта)
  • Отозвать согласие на обработку персональных данных
  • Отключить уведомления и отвязать Telegram в любой момент

Для реализации ваших прав обратитесь к нам по контактам, указанным в разделе 8.

7. Файлы cookie и аналитика

Сервис использует:

  • Технические данные (localStorage) — JWT-токен для аутентификации, настройки темы и языка интерфейса. Это не cookie в классическом понимании, но выполняет аналогичную функцию
  • Яндекс.Метрика — сторонний аналитический сервис, который может устанавливать cookie для сбора агрегированной статистики посещений. Мы не используем данные Метрики для идентификации конкретных пользователей

8. Контактная информация

По всем вопросам, связанным с обработкой персональных данных, вы можете обратиться:

  • Email: support@bondai.ru

Мы обязуемся рассмотреть ваше обращение в течение 30 дней с момента получения.
